# discordmd
cli application to parse and convert exported discord json to md

## Requirements
- Java 21 (used via Gradle toolchains)
- Gradle wrapper (included)

## Build
Run the Gradle wrapper to download dependencies and build the application:

```bash
./gradlew build
```

## Run
Use the application plugin entry point to print the greeting and example JSON payload:

```bash
./gradlew run
```
